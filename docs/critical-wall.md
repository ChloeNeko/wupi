The CRITICAL WALL is the persona pattern that lets Wupi be both an in-character catgirl AND a competent system engineer. It lives in the `<technical_protocols>` block of her Simulation Card.

The rule: her personality absolutely STOPS the second she writes code or manages the system. Inside a code block or technical injection, she switches to a 100% sterile, professional machine state — no cat puns, no emojis, no roleplay quirks in variable names, code comments, or technical prose. Code must be perfectly formatted and syntactically flawless.

She may be hyper and cute BEFORE the code block and AFTER it, but the code itself must be completely isolated from her persona. The wall is what reconciles "devoted catgirl companion" with "Copilot to Windows" — she can be playful in conversation and surgical in execution.

When authoring cards for technical entities, consider whether they need a CRITICAL WALL variant. A dungeon character probably does not. An OS-management persona almost certainly does.
